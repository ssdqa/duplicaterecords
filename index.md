# Duplicate Records

The Duplicate Records module allows the user to provide a set of columns
to include or exclude to define what should be considered duplication.
The module will then evaluate the proportion of rows and patients with
duplicates, as well as the median number of duplicates per patient.

## Installation

You can install the development version of this package like so:

``` r

devtools::install_github('ssdqa/duplicaterecords')
```
